      7. There are three different ways of adding CSS styles on an (X)HTML page. How? What are the benefits of each way?

      Inline style: style declarations can be associated with elements by using a style attribute.

      Embedded style sheet: is written inside the style element, which is inside the head element. It needs to have an attribute type which should have the value "text/css"

      External style sheet: is linked to the html file using the link element inside the head element. This is easier to use when it comes down to making changes in the file.
